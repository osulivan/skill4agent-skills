# skill4agent-skills

skill4agent 的技能集合。

## 什么是 skill4agent？

skill4agent 是一个收集精选技能的平台，专为中文用户设计。我们提供技能的中文翻译，并进行脚本安全检查。

## 这个仓库是什么？

这是 skill4agent 的官方技能仓库。目前包含 `search-install-skill-zh` 技能，提供两种搜索和安装技能的方式：
- npx 命令（需要 Node.js）
- API 调用（无需 Node.js）

支持使用中文搜索技能，对中文用户更加友好。

## 什么是技能？

技能是模块化包，通过专业知识、工作流和工具集成来扩展 AI 的能力。
